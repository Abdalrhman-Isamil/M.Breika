# موقع فضيلة الشيخ محمد أبو بريكة | معلم وقارئ القرآن الكريم

موقع تعريفي احترافي (Portfolio) ذو صفحة واحدة بتصميم إسلامي فاخر وعصري، مخصص لمعلم ومحفظ القرآن الكريم فضيلة الشيخ محمد أبو بريكة. 

الموقع مبني بالكامل باستخدام التقنيات الأساسية (HTML5, CSS3, Vanilla JS) بدون استخدام أي إطارات عمل (Frameworks) لضمان أقصى سرعة وأداء، وهو متوافق تماماً مع الهواتف الذكية والأجهزة اللوحية والمكتبية (Fully Responsive).

---

## 🌟 الميزات الأساسية للموقع

1. **شاشة ترحيبية فاخرة (Intro Screen):**
   - تأثير فتح المصحف الشريف (Mushaf Opening Effect).
   - عرض آية قرآنية كريمة بخط النسخ (Amiri Quran) مع إضاءة ذهبية متحركة.
   - نظام جزيئات ذهبية متطايرة (Canvas Particle System) في الخلفية.
   - تشغيل تلقائي لتلاوة عطرة للقرآن الكريم مع إمكانية كتم الصوت وتجاوز المقدمة.

2. **التصميم الإسلامي الفاخر (Premium Islamic Design):**
   - استخدام لوحة ألوان مستوحاة من الهوية الإسلامية الفاخرة (التركوازي الداكن، الذهبي، والأوف وايت).
   - استخدام زخارف إسلامية ونقوش هندسية في الخلفية.
   - بطاقات زجاجية عصرية (Glassmorphism Cards) مع تأثيرات إضاءة وحركة تفاعلية خفيفة (3D Tilt Effect).

3. **الأقسام الرئيسية:**
   - **الرئيسية (Hero Section):** تتضمن حديثاً شريفاً عن فضل تعلم القرآن، واسم الشيخ واللقب بشعار ذهبي مميز.
   - **الإحصائيات (Stats Section):** عداد رقمي تفاعلي لحساب (الطلاب المتميزين، حفظة القرآن، الختمات القرآنية، وسنوات الخبرة).
   - **نبذة عن الشيخ (About Section):** سيرة ذاتية للشيخ ومميزاته مع بطاقة تواصل سريعة.
   - **الخدمات (Services):** تفاصيل الحلقات والدروس المقدمة (حلقات الأطفال، الكبار، تصحيح التلاوة، التجويد، والتعليم عن بعد).
   - **المؤهلات والإجازات (Qualifications):** خط زمني تفاعلي يعرض إجازات الشيخ وسنده المتصل بالنبي ﷺ.
   - **آراء الطلاب (Testimonials):** شريط متحرك (Slider) يعرض شهادات الطلاب وأولياء الأمور بالشيخ.
   - **التواصل (Contact Section):** طرق التواصل المباشرة (رقم الهاتف، واتساب المباشر، وموقع تقريبي على الخريطة).

4. **تسهيل التواصل والوصول:**
   - زر واتساب عائم (Floating WhatsApp) مع رسالة مسبقة التعبئة للطلب المباشر.
   - زر العودة للأعلى (Scroll To Top).
   - دعم نسخ رقم الهاتف بمجرد النقر المزدوج عليه مع إشعار فوري (Toast Notification).

## 🔧 إرشادات إعداد وتشغيل الموقع

### 1. تشغيل الموقع محلياً (Running Locally)
بما أن الموقع يعتمد على ملفات JavaScript ووظيفة جلب الصوت وتشغيل الـ Canvas، يُنصح بتشغيل الموقع باستخدام خادم محلي (Local Web Server) بدلاً من فتح ملف `index.html` مباشرة في المتصفح لتجنب قيود الحماية (CORS).
- **باستخدام VS Code:** يمكنك استخدام إضافة **Live Server**.
- **باستخدام Node.js:** قم بتشغيل الأمر التالي في مجلد المشروع:
  ```bash
  npx serve
  ```
- **باستخدام Python:** قم بتشغيل الأمر التالي:
  ```bash
  python -m http.server 8000
  ```
  ثم افتح `http://localhost:8000` في متصفحك.

### 2. إضافة الملف الصوتي للشيخ (Audio Configuration)
لكي تعمل التلاوة الصوتية في شاشة الترحيب بصوت الشيخ محمد أبو بريكة نفسه:
1. قم بتسجيل أو تجهيز مقطع صوتي قصير للتلاوة بصيغة `MP3` (يُفضل أن تكون تلاوة لآية سورة الإسراء: *"إن هذا القرآن يهدي للتي هي أقوم..."* أو تلاوة هادئة ومميزة).
2. قم بتسمية الملف `recitation.mp3`.
3. ضعه داخل المجلد التالي: `assets/audio/`.

### 3. إعداد خريطة جوجل (Google Maps Embed)
الموقع يحتوي حالياً على تصميم بديل أنيق ومميز للخريطة (Map Placeholder). إذا كنت ترغب في دمج خريطة حقيقية تفاعلية لموقع الشيخ:
1. اذهب إلى خرائط جوجل وابحث عن الموقع المطلوب.
2. اضغط على مشاركة (Share) ثم تضمين خريطة (Embed a map) وانسخ الرابط الموجود داخل الـ `src` في كود الـ iframe.
3. افتح ملف [main.js](file:///d:/All_Folders_Frrom_DESKTOP/Sheikh-Mohamed-Breka/js/main.js) وقم بإلغاء التعليق عن دالة `initMap` (السطر 240) وقم باستدعائها في دالة البوت `boot` (السطر 384)، مع وضع الرابط الخاص بك.
